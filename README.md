# Threat Hunting
This repository is used to store scripts, notebooks, and resources generated by Target's Threat Hunting team.
You can find scripts pertaining to each technique or goal in their relevant subdirectories.

# Contributor Instructions
The .ipynb files can be opened and edited directly with Jupyter Notebook.  Be sure to restart the kernel before any 
new commits, which will ensure that any data or output is committed with the code.
